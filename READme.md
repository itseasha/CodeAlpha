URL Shortener

A simple, fast, and efficient URL shortening service built with Node.js, Express, and SQLite. Shorten long URLs, track clicks, and manage your links with ease.

Features

**Shorten URLs** - Convert long URLs into short, shareable links
**Click Tracking** - Monitor how many times each short link is accessed
**Copy to Clipboard** - One-click copy of shortened URLs
**Delete Links** - Remove unwanted short URLs
**Responsive Design** - Works perfectly on desktop, tablet, and mobile
**Fast Redirects** - Instant redirection to original URLs
**Persistent Storage** - SQLite database stores all your links

Tech Stack

| Technology | Purpose |

| **Node.js** | JavaScript runtime |
| **Express.js** | Web framework |
| **SQLite3** | Lightweight database |
| **Nanoid** | Unique short code generation |
| **HTML/CSS/JS** | Frontend interface |

 Prerequisites

[Node.js](https://nodejs.org/) (v14 or higher)
npm (comes with Node.js)
